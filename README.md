Example:

```zsh
cargo run -- -b canpute -i can0 -n 06
```
This assumes you make a directory in the current folder called /binaries and put the desired binary in there. Otherwise, specify the path of the target binary.

If you are looking for the device implementation, please see [/ezfd-bootloader-device](https://github.com/schommie/ezfd-bootloader-device)
